# 実データ（市況資料）の反映計画

ご提供いただいた市況報告書の画像に基づき、プロトタイプのダミーデータを実データへ置き換え、継続的な運用を可能にします。

## 2. 継続的な入力方法の提供
画像（FAXやPDF）のデータを手軽にグラフにするため、以下の2段階で進めます。

1. **Excel/CSVテンプレートの作成**: 
   - 今回の資料の項目に合わせたシンプルな入力表（Excel等で開ける形式）を作成します。
2. **自動反映機能の強化**:
   - そのファイルを保存してスクリプトを実行するだけで、グラフが最新の状態に更新されるようにします。

## 3. ユーザー様への提案
## 4. 焼津漁協サイト（msinfo）からの自動取得
提供いただいた [焼津漁協相場情報](https://www.yaizu-gyokyo.or.jp/itiba/msinfo/) からの取得を検討します。
- **取得内容**: 「旋網冷凍かつお（海外旋網船）」のサイズ別（4.5/2.5/1.8kg上）の高値・安値・水揚量。
- **実装**: Pythonスクリプトで最新の相場情報を定期的にチェックし、CSVに自動追記するロジックを検討。

## 5. 水揚量の可視化（棒グラフ追加）
価格の折れ線グラフに加え、その日の水揚量を棒グラフで表示します。
- **表示形式**: Chart.jsの「混合グラフ（折れ線＋棒グラフ）」または「上下2段構成」。
- **メリット**: 「水揚量が多い時に価格が下がる」といった相関関係が視覚的に理解しやすくなります。
