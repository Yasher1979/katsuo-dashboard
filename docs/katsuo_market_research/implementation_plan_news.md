# カツオ関連ニュース表示機能の実装計画

カツオ漁業や市場に関する最新トピックをダッシュボード上でまとめて確認できる機能を追加します。

## 1. 概要
- **目的**: 相場データだけでなく、その背景にあるニュース（漁況、燃費油、海況情報など）をひと目で確認できるようにする。
- **表示場所**: メインタブに「トピック」または「ニュース」タブを新設。

## 2. 実装内容

### UI/UX (Frontend)
- **新タブ의 追加**: `index.html` にニュース表示用のタブアイテムとビューコンテナを追加。
- **ニュースカードのデザイン**: ニュースのタイトル、日付、ソース名、および概要を表示するカード型レイアウト。
- **外部リンク連携**: タイトルをタップすると、情報元のオリジナルソース（Webページ）を開く仕組み。

### データ構造 (Data)
- **JSON形式での管理**: `data/katsuo_news.json` を新設。
- **項目の例**:
  ```json
  [
    {
      "date": "2026-02-18",
      "title": "海外まき網漁船の新着漁況情報",
      "source": "JAFIC",
      "url": "https://www.jafic.or.jp/...",
      "category": "漁況"
    }
  ]
  ```

## 3. ニュースソースの候補
- **水産庁**: 報道発表（生産対策、規制情報など）。
- **JAFIC (漁業情報サービスセンター)**: 漁況・海況速報。
- **さかなのNEWS / 各地漁協のブログ**: 現場のリアルタイムトピック。

## 4. ステップ
1.  **JSONファイルの作成**: サンプルのニュースデータを数件分作成。
2.  **HTML/CSSの更新**: タブのUIとニュース表示エリアを作成。
3.  **JavaScriptの更新**: ニュースデータを読み込み、自動的にレンダリングするロジックを `dashboard.js` に追加。

---
> [!NOTE]
> 自動取得（スクレイピング）については、ソース側の規約を考慮し、まずは「管理用のJSONファイルを更新すると画面に反映される」という確実な仕組みから構築します。
